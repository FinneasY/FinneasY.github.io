无期限模型与世代交叠模型

发表于2025-04-25|更新于2025-04-26|宏观经济学
|总字数:759|阅读时长:2分钟|浏览量:|评论数:

前言

  • 本章所讨论的两个模型与Solow模型有一个显著区别，即这两个模型中总体经济的动态变化是由微观层面的决策决定的。
  • 这两个模型仍然把劳动和知识的增长率看作给定的，但资本存量的变动方程来自竞争性市场上家庭最大化行为与厂商最大化行为的交互作用，因此储蓄率不再是外生的，也不必保持不变的。

Ramsey-Cass-Koopmans模型

无期限模型

模型假设

生产技术

  生产函数为F(K,AL)F(K,AL)F(K,AL)，其中F()˙F(\dot)F()˙​满足Solow模型中的所有假设。为了方便起见，假设没有折旧。

K˙(t)=Y(t)−ζ(t)\dot{K}(t) = Y(t) - \zeta (t) K˙(t)=Y(t)−ζ(t)

  其中ζ(t)\zeta(t)ζ(t)是总消费。CCC用来表示人均消费，ccc则表示单位有效劳动平均消费。

厂商

  • 生产函数为F(K,AL)F(K,AL)F(K,AL)
  • 要素市场和商品市场都是竞争的
  • AAA的增长率为ggg，外生给定
  • 厂商都归家庭所有，其利润最终全部归于家庭

家庭

  家庭效用函数形式如下：

U=∫t=0∞e−ρtu(C(t))L(t)Hdt.U = \int_{t=0}^{\infty} e^{-\rho t} u(C(t)) \frac{L(t)}{H} dt. U=∫t=0∞​e−ρtu(C(t))HL(t)​dt.

  • C(t)C(t)C(t)是每个家庭成员在时间ttt的消费
  • u()˙u(\dot)u()˙​是瞬时效用函数，表示给定时刻下每个家庭成员的效用
  • L(t)L(t)L(t)是经济中的总人口，所以L(t)H\frac{L(t)}{H}HL(t)​是家庭成员数量
  • u(C(t))L(t)/Hu(C(t))L(t)/Hu(C(t))L(t)/H就是家庭所有成员在时间ttt的总瞬时效用
  • ρ\rhoρ是折现率，rhorhorho越大，则家庭就越看重当期消费而不是未来消费。
      瞬时效用函数形式如下：

u(C(t))=C(t)1−θ1−θ,θ>0,ρ−n−(1−θ)g>0u(C(t)) = \frac{C(t)^{1-\theta}}{1-\theta}, \theta > 0, \rho-n-(1-\theta)g > 0 u(C(t))=1−θC(t)1−θ​,θ>0,ρ−n−(1−θ)g>0

  • 这一效用函数形式通常被称为相对风险规避系数不变（CRRA）的效用函数。θ\thetaθ是相对风险规避系数，与CCC无关。这样的效用函数形式可以保证经济向平衡增长路径收敛。
  • θ\thetaθ决定了家庭在不同时期之间调整消费的意愿，θ\thetaθ越小，则消费上升所引起的边际效用下降就越慢，家庭就愿意接受消费的跨期变化。
