1. 索洛模型 (Solow Model)

1.1. 经济增长的一些基本事实

  • 增长奇迹：
    一个国家的增长在长期中远远高于世界平均水平，从而导致该国在世界收入分布中的排名迅速上升。

  • 增长灾难：
    一个国家的增长远远低于世界平均水平的现象。

  • 研究经济增长的最终目的：

    • 探索是否可以全面加快增长，或者使穷国的生活水平更接近先进国家。
    • 索洛模型的基本结论：
      • 不论是人均产出随着时间大幅增长，还是人均产出在不同地区存在巨大差异，都难以用实物资本的积累来解释。

1.2. 假设

投入与产出

关于生产函数的假定

Y(t)=F(K(t),A(t)L(t))Y(t) = F(K(t), A(t)L(t)) Y(t)=F(K(t),A(t)L(t))

  1. YYY 是产出，KKK 是资本，AAA 是知识（劳动效率），LLL 是劳动。
  2. ttt 不直接进入函数，而是通过 KKK, LLL, AAA$ 进入。
  3. 劳动增强型/哈罗德中性技术进步：
    ALALAL 以乘积形式进入函数，称 ALALAL 为有效劳动。
  • 假设：

    • 规模报酬不变：

      F(cK,cAL)=cF(K,AL),∀c≥0F(cK, cAL) = cF(K, AL), \quad \forall c \geq 0 F(cK,cAL)=cF(K,AL),∀c≥0

      • 说明：
        1. 经济规模已经足够大，使得专业分工的好处已得到最大限度的利用。
        2. 相对于资本、劳动和知识来说，其他投入要素并不重要。

    • 稻田条件 (Inada, 1964)：

      lim⁡k→0f′(k)=∞,lim⁡k→∞f′(k)=0\lim_{k \to 0} f'(k) = \infty, \quad \lim_{k \to \infty} f'(k) = 0 k→0lim​f′(k)=∞,k→∞lim​f′(k)=0

Intensive Form （由规模报酬不变推出）

F(KAL,1)=1ALF(K,AL)F\left(\frac{K}{AL}, 1\right) = \frac{1}{AL}F(K, AL) F(ALK​,1)=AL1​F(K,AL)

  • 定义：
    • k=KALk = \frac{K}{AL}k=ALK​：单位有效劳动的平均资本量。
    • y=YALy = \frac{Y}{AL}y=ALY​：单位有效劳动的平均产出。
  • 简化形式：

    y=f(k)y = f(k) y=f(k)

    • 假定 f(0)=0f(0) = 0f(0)=0, f′(k)>0f'(k) > 0f′(k)>0, f′′(k)<0f''(k) < 0f′′(k)<0。
      即资本边际产出为正，并且它随资本量增加而递减，保证经济路径不会发散。
  • Cobb-Douglas 函数满足上述所有条件。

关于投入要素变化的假定

  1. 时间的连续性：模型中假定时间是连续的。

  2. 资本、劳动和知识的初始值：
    L(0)>0,A(0)>0,K(0)>0L(0) > 0, A(0) > 0, K(0) > 0L(0)>0,A(0)>0,K(0)>0，且 LLL 和 AAA 分别按固定比率增长：

    L˙(t)=nL(t),A˙(t)=gA(t)\dot{L}(t) = nL(t), \quad \dot{A}(t) = gA(t) L˙(t)=nL(t),A˙(t)=gA(t)

    • 其中 nnn 和 ggg 是外生参数。
    • L(t)L(t)L(t) 和 A(t)A(t)A(t) 分别按指数增长：

      L(t)=L(0)ent,A(t)=A(0)egtL(t) = L(0)e^{nt}, \quad A(t) = A(0)e^{gt} L(t)=L(0)ent,A(t)=A(0)egt

  3. 资本动态方程：
    总产出用于消费和投资，其中投资等于 $$Y(t) - C(t)$$，资本的变化由以下方程决定：

    K˙(t)=sY(t)−δK(t),n+g+δ>0\dot{K}(t) = sY(t) - \delta K(t), \quad n + g + \delta > 0 K˙(t)=sY(t)−δK(t),n+g+δ>0

    • 其中 sss 为储蓄率，δ\deltaδ 为折旧率。

1.3. Solow 模型的动态变化

kkk 的动态变化

  • 对 kkk 求导：

    k˙(t)=sf(k(t))−(n+g+δ)k(t)\dot{k}(t) = sf(k(t)) - (n + g + \delta)k(t) k˙(t)=sf(k(t))−(n+g+δ)k(t)

  • 单位有效劳动平均资本存量的变化率由两项之差决定：

    • 实际投资：sf(k)sf(k)sf(k)。
    • 所需持平投资：(n+g+δ)k(n + g + \delta)k(n+g+δ)k。

  • 相图分析：

    • 当 sf(k)>(n+g+δ)ksf(k) > (n+g+\delta)ksf(k)>(n+g+δ)k，即 k˙>0\dot{k} > 0k˙>0，kkk 上升。
    • 当 sf(k)<(n+g+δ)ksf(k) < (n+g+\delta)ksf(k)<(n+g+δ)k，即 k˙<0\dot{k} < 0k˙<0，kkk 下降。

平衡增长路径

  • 当 k=k∗k = k^*k=k∗ 时，经济达到稳定状态。此时：
    • AAA 的增长率为 ggg，LLL 的增长率为 nnn。
    • KKK 和 YYY 的增长率为 n+gn + gn+g：

      K˙K=n+g,Y˙Y=n+g\frac{\dot{K}}{K} = n + g, \quad \frac{\dot{Y}}{Y} = n + g KK˙​=n+g,YY˙​=n+g

    • 人均资本 KL\frac{K}{L}LK​ 和人均产出 YL\frac{Y}{L}LY​ 的增长率为 ggg：

      K˙/LK/L=g,Y˙/LY/L=g\frac{\dot{K}/L}{K/L} = g, \quad \frac{\dot{Y}/L}{Y/L} = g K/LK˙/L​=g,Y/LY˙/L​=g

  • 平衡增长路径上，人均产出的增长率由技术进步率 ggg 唯一决定。

1.4. 储蓄率变化的影响

对产出的影响

  • 储蓄率 sss 增加：

    • 实际投资 sf(k)sf(k)sf(k) 增加。
    • 均衡单位有效劳动资本量 k∗k^*k∗ 上升。
    • 均衡单位有效劳动平均产出 f(k∗)f(k^*)f(k∗) 增加。

  • 人均产出 YL\frac{Y}{L}LY​ 的增长率：

    YL=Af(k),Y˙/LY/L=g\frac{Y}{L} = Af(k), \quad \frac{\dot{Y}/L}{Y/L} = g LY​=Af(k),Y/LY˙/L​=g

  • 结论：
    储蓄率的变化只有水平效应，没有增长效应。改变了经济平衡增长路径的位置，但不影响人均产出的增长率，该增长率仅由技术进步率 ggg 决定。

对消费的影响

  • 平衡增长路径上的消费量：

    c∗=f(k∗)−sf(k∗)=f(k∗)−(n+g+δ)k∗c^* = f(k^*) - sf(k^*) = f(k^*) - (n+g+\delta)k^* c∗=f(k∗)−sf(k∗)=f(k∗)−(n+g+δ)k∗

    • 消费达到最大时，对应的 k∗k^*k∗ 称为资本存量的黄金律水平。

1.5. 定量意义

储蓄率对产出的长期影响

  • 储蓄率上升对长期产出的弹性：

    ∂y∗/y∂s/s=αk∗1−αk∗\frac{\partial y^* / y}{\partial s / s} = \frac{\alpha_{k^*}}{1 - \alpha_{k^*}} ∂s/s∂y∗/y​=1−αk∗​αk∗​​

    • 其中 αk∗=k∗f′(k∗)f(k∗)\alpha_{k^*} = \frac{k^*f'(k^*)}{f(k^*)}αk∗​=f(k∗)k∗f′(k∗)​，表示资本收入占总收入的比例。

收敛速度

  • 根据泰勒展开式：

    k˙≃−λ(k−k∗),λ=(n+g+δ)[1−αk(k∗)]\dot{k} \simeq -\lambda (k - k^*), \quad \lambda = (n+g+\delta)[1 - \alpha_k(k^*)] k˙≃−λ(k−k∗),λ=(n+g+δ)[1−αk​(k∗)]

    • λ\lambdaλ 表示收敛速度。
